Program Notes

How to maintain this dashboard.

Update data at the top of the file
Edit projectData.tasks, projectData.risks, and projectData.milestones. The dashboard updates automatically.
Change a task status
Set status to one of: Not Started, In Progress, Blocked, or Complete.
Add a new risk
Add an object to projectData.risks with risk name, severity, owner, mitigation, and status. Open risk counts will update automatically.
Set overall status
Update projectData.meta.overallStatus to Green, Yellow, or Red.
